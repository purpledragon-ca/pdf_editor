Usage Tips

  • Page numbers start from 1.
  • Split example: 1-3, 5, 7-9.
  • Reorder supports reverse order: e.g., 10-7 means 10,9,8,7.
  • Encryption: If owner password is not provided, it defaults to the user password.
  • For scanned documents/images, text extraction may be empty.
  • Password-protected PDFs need to be decrypted first before processing.
